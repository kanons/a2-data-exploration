A2: Data Exploration

The following visualization shows the death rates of impaired driving from 2014 by State. Source: Data.gov

The data is composed of the rate of deaths by age and gender (per 100K population) for people killed in crashes involving a driver with BAC greater than or equal to 0.08%. States without a bar indicate supressed data where fatality rates based on fewer than 20 deaths were suppressed.

Data Expression and Effectiveness

For the visual layout, I chose to display the data in a bar chart because the dataset included discrete variables, the states, and the individual bars would best express the data effectively. The visual encodings used within the bar chart are length, position, and area. The length shows the value of the death rate where the solid-fill bars make it easier to compare the length of the bars with each other. The position of each data point is emphasized through transitions where the user is able to see the position of the top of the bars change between charts. This encoding effectively shows the difference of value between each control option. Lastly, area in this visual layout is used to express the drastic changes in the chart. Because the chart includes suppressed data, as the controls change, some of the bars disappear. By using bars that fill an area of the chart, the changes in which states have no bar (and no area) are easier to identify. The states are kept in alphabetical order to make it easier to keep track of the changes between charts, and a hover text shows the exact value to express more information that is not encoded in the chart itself. The controls chosen were buttons on the right side of the graph where all features are grouped into a button group. This is to keep the interaction intuitive because only one feature can be selected at a time, instead of being able to select multiple features at once. All of the data columns from the dataset are listed within the button group to express as much of the data as possible and allow the user to select data of their interest. The charts are shown one-at-a-time to best express the values of the dataset and avoid confusion. Due to the gaps within the data (from the suppressed data), if multiple features were shown at once, it would visually occlude the importance of these gaps. Therefore, showing a value at a time for each state, replacing the previous bar, best shows the transitions of value between the features and keeps the importance of the ‘blank’ data points.

Control Elements: Explained

The controlling elements used are different age groups and gender of the death rates. The ‘All Ages’ control gives the user an overview of the impaired driving death rates across the country. The wide range enables the user to ask questions regarding the overall trend of death rates across the country, with the option of asking detailed questions about each state. Some questions for this control could be “Which state has the highest rate” or finding data relatable to them such as “How does Washington State compare with other states?” Each of the following ‘age’ controls evokes questions regarding the comparison between and within each age group. For ‘0-20’, would there be lower rates? The ‘21-33’ and ‘35+’ age group controls can lead the user to question the correlation between legal drinking age and death rates. Because the topic is related to age due to the restrictions for driving and drinking, providing controlling elements to view the data for different age groups enables the user to compare and contrast the data using these controls. For example, through using the tools, I have found that the death rates for older groups (‘21-33’ and ‘35+’) are higher in comparison to the ‘0-20’ group. The gender controls also provide insight into the differences in death rates between the two groups, enabling the user to ask questions that discover how the rates of the two differ. For example, they can ask “Which state has the highest death rate for males?” or “Which gender has the lowest rates overall?” By using the ‘male’ and ‘female’ controls, it can be found that males have higher death rates for impaired driving. In addition, throughout these controls, the user can also focus on certain states and compare the rates for that state which enables the user to different questions regarding the details of that portion of the data.
